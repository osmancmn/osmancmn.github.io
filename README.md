# Osman Çimen Portfolyo

Bu paket aşağıdaki dosyaları içerir:

- index.html
- style.css
- script.js

## Çalıştırma

1. Klasörü çıkar.
2. `index.html` dosyasına çift tıkla.
3. Site tarayıcıda açılır.

## Not

- Proje kartları `script.js` içinden dinamik olarak üretilir.
- Tema butonu ile koyu/açık mod değiştirilebilir.
- Mobil menü ve proje filtreleme aktiftir.
